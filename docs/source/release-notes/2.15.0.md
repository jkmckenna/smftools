(v2.15.0)=

2.15.0 2026-07-19
Version 2.15.0 adds opt-in secondary-alignment rescue for nested/overlapping reference variants (`rescue_secondary_alignments`), fixes a pysam direct-signal probability inversion bug, parallelizes the modkit TSV backend, and streams HMM/spatial task outputs to disk instead of holding them in memory. It also closes out a real production out-of-memory investigation on a multi-hundred-thousand-read batch run: raw ingestion no longer accumulates a whole chromosome or lets process-pool submissions outrun consumption, the per-worker memory watchdog gained tolerance/grace-period handling and a retry-with-fewer-workers fallback instead of aborting a whole task list over a single killed worker, and `materialize()`'s ragged-to-dense conversion was rewritten to stream shard-by-shard with narrower on-disk dtypes and pushed-down read filtering -- cutting a single preprocess task's peak memory from 16GB to under 3GB. Clustermap plotting gained a configurable per-plot read cap (`clustermap_max_reads_per_plot`, default 5000) with reproducible subsampling, and its own process pools are now covered by the same memory watchdog. Also fixes a barcode-sidecar collapse when `skip_bam_split=True` lacks BC/bi tags on the aligned BAM, a `reindexing_offsets` crash from a stray YAML null, and several docs/CI build failures.
