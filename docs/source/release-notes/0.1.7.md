(v0.1.7)=

0.1.7 2025-07-03
Version 0.1.7 added model-interpretability tooling (SHAP, plus interpretability modules for both sklearn and torch.nn models), GradCAM support for CNNs, and PR/ROC metrics with a more flexible CNN architecture. Sliding-window train/test/inference wrappers for both Lightning and sklearn backends were added, along with broader training, inference, and post-inference evaluation updates.
