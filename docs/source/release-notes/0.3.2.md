(v0.3.2)=

0.3.2 2026-02-03
Version 0.3.2 introduced the `chimeric` and `variant` CLI pipeline stages with dedicated plotting, built out cross-sample PCR chimera detection on top of the zero-Hamming span work from 0.3.1 (merge-gap options, overlap modes, NaN run limits), and reorganized plotting code into workflow-specific modules. Latent-space plotting gained PCA explained-variance and component plots with grid embeddings, and var-mask support was added across PCA/NMF/CP. Mismatch-frequency plotting gained base-frequency panels and ambiguous GpC/CpG site handling. This version also added the repo's AGENTS.md/CLAUDE.md agent instruction files and fixed a number of unit/smoke tests.
